#  Set up your wallet
This guide will help you set up your wallet to start using Taiko.

## Prerequisites
- You have a wallet installed (e.g., [Rabby](https://rabby.io/), [MetaMask](https://metamask.io/)).

## Devnet
### Add chains to your wallet

Click **Connect Wallet** at the links below:

- Super World Computer devnet: https://chainlist.org/chain/167 (TODO:167 is placeholder)

### Add devnet tokens to your wallet

Find the token you want to import on the [devnet addresses](/network-reference/contract-addresses#devnet) page and use the address to add it to your wallet.

### Troubleshooting

#### The add buttons did not work

If the add buttons did not work, head over to the [RPC configuration](/network-reference/rpc-configuration#devnet) and [devnet addresses reference](/network-reference/contract-addresses#devnet) pages and add them to your wallet manually.
